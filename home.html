HTML File Structure HTML

The basic structure of any HTML page consists of the following elements:> ,> ,> and> .: <html>, <head>,and<body>.

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
</head>
<body>
  <h1>Hello World!</h1>
</body>
</html>

Headings

Headings are used to define the importance of text, from <h1> (largest) to <h6> (smallest).

<h1 style="color: red;">Hello World! 1</h1>
<h2 style="color: blue;">Hello World! 2</h2>
<h3 style="color: green;">Hello World! 3</h3>
<h4 style="color: purple;">Hello World! 4</h4>
<h5 style="color: orange;">Hello World! 5</h5>
<h6 style="color: brown;">Hello World! 6</h6>

Line Break

The <br> tag creates a line break without starting a new paragraph.

Line 1<br>Line 2

Line 1
Line 2

Lists

<ul> creates an unordered list (bullets), and <ol> creates an ordered list (numbers).

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>
  • Item 1
  • Item 2
  1. Step 1
  2. Step 2

Tables

<table> creates a table, and border, bordercolor, bgcolor add styling.

<table border="1" bordercolor="blue" bgcolor="lightyellow">
  <tr><td>Cell 1</td><td>Cell 2</td></tr>
  <tr><td>Cell 3</td><td>Cell 4</td></tr>
</table>
Cell 1Cell 2
Cell 3Cell 4

Images

<img> adds an image, and height, width, border, bordercolor add styling.

<img src="img/logo.jpg" height="100" width="100" border="2" bordercolor="green">

Video/Audio

<iframe> adds external content like videos, and width, height define the size.

<iframe width="300" height="200" src="https://www.youtube.com/watch?v=6QAELgirvjs>&list=PLDoPjvoNmBAw_t_XWUFbBX-c9MafPk9ji"></iframe>

Text Formatting

These tags are used for text formatting: <p>, <b>, <u>, <i>, <sub>, <sup>, <big>, <small>.

<p>This is <b>bold</b>, <u>underlined</u>, <i>italic</i>, H<sub>2</sub>O, X<sup>2</sup>, <big>big</big>, and <small>small</small>.</p>

This is bold, underlined, italic, H2O, X2, big, and small.
